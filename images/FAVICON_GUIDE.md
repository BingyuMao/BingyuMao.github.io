# Favicon Emoji Guide

## How to Change Emoji

The current favicon is an emoji! To change it:

1. Open `images/favicon.svg` in any text editor
2. Find the line with `<text>⭐</text>`
3. Replace `⭐` with any emoji you like
4. Save the file
5. Clear your browser cache and refresh to see the change

## Emoji Suggestions

- 🌸 Cherry blossom
- ✨ Sparkles
- 🌟 Glowing star
- ⚡ Lightning
- 💎 Gem
