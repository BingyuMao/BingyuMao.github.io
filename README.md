# Bingyu Mao's Personal Website

Personal academic and professional website built with Jekyll.

## Local Development

To run the site locally:

```bash
bundle install
bundle exec jekyll serve
```

The site will be available at `http://localhost:4000`.

## Structure

- `_pages/`: Main content pages (about, publications, research, CV)
- `_sass/`: Styling and theme customization
- `_includes/`: Reusable HTML components
- `_layouts/`: Page layouts
- `files/`: PDFs and downloadable files
- `images/`: Site images

## Deployment

The site is automatically deployed to GitHub Pages when changes are pushed to the main branch.
